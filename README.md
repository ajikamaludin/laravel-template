## Requirements

- PHP 8.1 or latest
- Node 16+ or latest

## How to run

```bash
cp .env.example .env # configure app for laravel
touch database/database.sqlite # if you use .env.example with default sqlite database
composer install
npm install
npm run dev # compiling asset for development
```
